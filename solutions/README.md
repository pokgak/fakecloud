# Answer key

Worked solutions for every chapter in `examples/`. No peeking until your
own plan output disagrees with the chapter's "check yourself" section —
the marks you draw yourself stick better.

Each directory mirrors its chapter and contains the finished, verified
Terraform (optional missions like the `ignore_changes` truce or the
`create_before_destroy` footgun are left as commented blocks to toggle).
Fill in your own playground's sandbox id in the provider blocks, then
`terraform init` — the provider comes from the registry.

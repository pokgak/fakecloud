How it works

1. Create a playground — it's your own private fakecloud.
2. Point Terraform at it: the provider's sandbox attribute (the dashboard shows the exact block to copy).
3. Work through the chapters in examples/ — you write the Terraform, the dashboard shows the consequences.
4. Playgrounds idle for 30 days fade away like all good pretend infrastructure.
